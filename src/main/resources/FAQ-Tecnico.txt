¿Cómo organizar la sesion?

IMPORTANTE: ASEGURARSE DE ESTAR EN TU RAMA CUANDO EMPIEZAS A PROGRAMAR
INICIO:
1- Hacer un git pull de la rama main
2- Hacer git checkout para ir a tu rama 
3- Hacer un git merge main
FINAL:
1- git add .    para añadir todas las modificaciones
2- git commit -m "mensaje de commit"
3- git push
4- git checkout main
5- git merge nombreDeLaRama
6- git push
